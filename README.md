Spendly — Smart Finance Tracker

Spendly is a modern expense and income tracking web application designed with a focus on clean UI, smooth interactions, and an intuitive user experience. It enables users to manage finances efficiently while providing meaningful insights into spending and saving behavior.

Overview

This project is a single-page web application built using HTML, CSS, and JavaScript. It operates entirely on the client side and uses browser local storage for data persistence, making it lightweight and easy to deploy.

Features
Expense Management
Add, categorize, and delete expenses
Category-based organization (Food, Transport, Shopping, etc.)
Card-based transaction interface
Income Tracking
Record multiple income sources (salary, freelance, business, etc.)
Monthly income aggregation
Separate income overview section
Dashboard
Net balance calculation (income vs expenses)
Real-time updates
7-day spending trend visualization
Category-wise spending distribution
Insights and Analytics
Savings rate calculation
Monthly comparison charts
Automated financial insights based on user data
Budget Tracking
Set a monthly budget
Track spending against budget
Visual indicators for budget usage
User Experience
Responsive design
Smooth transitions and micro-interactions
Animated charts and dynamic UI updates
Data Persistence
Uses browser localStorage
No backend or database required
Technology Stack
Frontend: HTML5, CSS3, JavaScript
Charting Library: Chart.js
Storage: Browser LocalStorage
Fonts: Inter, JetBrains Mono

Getting Started
Clone or download the repository
Open the main file in a browser:
spendly-v3.html

No additional setup or dependencies are required.

How It Works
Data is stored locally in the browser using localStorage
All calculations (income, expenses, savings) are performed dynamically
Charts are rendered and updated in real time based on user input
